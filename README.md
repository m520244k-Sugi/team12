## アプリ名
謎解き脱出ゲーム

## アプリ概要
プレイヤーが部屋の中を探索し、隠されたアイテムやヒントを集めながら様々な謎を解き、部屋からの脱出を目指すゲームです。アイテムを正しく使用したり、ヒントを組み合わせて仕掛けを解除したりすることでゲームを進めます。

## ゲームの流れ
1. タイトル画面を表示する。
2. ゲームを開始する。
3. 部屋を探索してアイテムやヒントを集める。
4. 謎を解いて鍵や仕掛けを解除する。
5. 出口を開いて脱出するとゲームクリアとなる。

## 主な機能
- 部屋の探索
- アイテムの取得・使用
- 謎解き
- ヒント表示
- ゲームクリア画面
